Fast ASP.NET websites

Welcome to FAST ASP.NET websites. This Github repository contains all the source code and files that you will need to follow along with in the book. Right from chapter 1, we take a slow, poorly performing website right up to chapter 12 where the website is optimized for speed and performance. The book covers essential tips and tricks that you can apply to your ASP.NET websites today.
